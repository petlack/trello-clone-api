# Trello clone API

## Features
- ⚙ Python, [Ariadne](https://ariadnegraphql.org/), [Flask](https://flask.palletsprojects.com/en/2.2.x/)
- 📝 GraphQL
- 📚 In-Memory database
- 🔓 CORS enabled

## Install
```bash
git clone git@github.com:petlack/trello-clone-api.git
cd trello-clone-api
pip install -r requirements.txt
```

## Development
```bash
./run.sh
```
